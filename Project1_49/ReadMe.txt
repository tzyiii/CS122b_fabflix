All *.class is in the directory of zip : java/bin

All *.java is in the directory of zip :java/src




To run: 

cd java/bin

java Menutest




To compile and run:

cd java/src  

javac *.java

java MenuTest



**Set java/mysql-connector-java-5.1.40-bin.jar as your classpath



CLASSPATH=.:$CLASSPATH:~/mysql-connector-java-5.1.40-bin.jar

export CLASSPATH
